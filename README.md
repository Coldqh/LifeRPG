## Обновление GitHub Pages

Распакуй содержимое архива в `C:\PrimeRPG` с заменой файлов.

```powershell
cd C:\PrimeRPG
git add .
git commit -m "PRIME RPG v2.0 elite interface"
git push
```

Открыть свежую версию:

```text
https://coldqh.github.io/LifeRPG/?v=2.0.0
```

На телефоне при необходимости: `Настройки → Обновить версию`.
