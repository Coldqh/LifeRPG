# PRIME RPG v1.4

Компактный habit tracker / RPG-система прокачки реальной жизни.

Работает как статическое приложение: `HTML + CSS + JavaScript`. Сервер не нужен. Данные хранятся в браузере через `localStorage` + резервный ключ backup.

## Что внутри

- профиль с XP, уровнем, неделей и статами;
- ежедневные квесты с эмодзи-иконками;
- недельные категории;
- штрафы;
- pending XP: опыт текущего дня/недели не применяется до закрытия;
- история дней и недель;
- импорт/экспорт JSON;
- импорт кастомных quest pack файлов;
- кнопка `Обновить версию` для телефона;
- in-app уведомление о новой версии;
- PWA/service worker с network-first стратегией.

## Обновление версии

Если на телефоне криво работают сохранения или висит старая версия:

1. Открой приложение.
2. Нажми `Обновить версию`.
3. Приложение сохранит данные, очистит PWA-кэш и перезагрузит свежие файлы.

История и текущие галочки не удаляются.

## Quest pack

Формат:

```json
{
  "type": "prime-rpg-quest-pack",
  "mode": "merge",
  "dailyQuests": [
    {
      "category": "BODY",
      "title": "BODY",
      "stat": "BODY",
      "items": [
        { "text": "мобилити 15 минут", "xp": 10, "stat": "BODY" }
      ]
    }
  ]
}
```

Режимы:

- `merge` — добавить квесты в категорию;
- `replace` — заменить квесты категории.

Максимум: 10 квестов на категорию.

## Важно

Данные лежат локально в браузере. Раз в неделю делай экспорт JSON.
