# PRIME RPG

Личная RPG-система прокачки реальной жизни.

Стек: `HTML + CSS + JavaScript`. Сервер не нужен. Данные хранятся локально в браузере через `localStorage`.

## Главное в v0.2

Теперь день не создаётся вручную.

- приложение живёт по московскому времени;
- текущий день создаётся автоматически;
- галочки и поля сохраняются сразу;
- в `00:00 МСК` активный день закрывается и попадает в историю;
- новый день создаётся автоматически;
- в ночь с воскресенья на понедельник закрывается неделя;
- недельная информация добавляется в общий отчёт;
- если приложение было закрыто, синхронизация догонит календарь при следующем открытии.

## Что работает

- ежедневные квесты;
- штрафы;
- live XP;
- ранги дня;
- общий уровень;
- статус-ранг;
- статы персонажа;
- недельные боссы;
- обязательный минимум недели;
- автоистория дней и недель;
- экспорт/импорт JSON;
- PWA manifest;
- service worker для базового offline-cache.

## Быстрый запуск локально

Открой `index.html` в браузере.

Лучше через локальный сервер:

```bash
python -m http.server 8080
```

Потом открыть:

```text
http://localhost:8080
```

## Хостинг на GitHub Pages

1. Загрузи все файлы в корень репозитория.
2. Проверь, что `index.html` лежит в корне.
3. Открой `Settings` → `Pages`.
4. Выбери:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Сохрани.

Сайт будет доступен по адресу вида:

```text
https://coldqh.github.io/LifeRPG/
```

## Важно

Данные лежат в браузере. Если почистить данные сайта — отчёты исчезнут.

Раз в неделю нажимай `Экспорт` и сохраняй backup JSON.
